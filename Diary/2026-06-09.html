Boris Cherny — Claude Code @anthropicai

多个 benchmark 显示 Opus 在长时间自主运行任务上是最佳模型。
让 Opus 自主运行数小时/天数的五个技巧：

  1. 使用 auto 权限模式，避免 Claude 每次都需要人工批准
  2. 使用 dynamic workflows，让 Claude 编排成百上千个 agent 完成任务
  3. 使用 /goal 或 /loop 提示 Claude 持续执行直到完成
  4. 在云端使用 Claude Code（桌面或手机 app 是最简单的方式），这样关笔记本也不会中断
  5. 确保 Claude 有端到端自我验证的能力：Chrome 浏览器扩展、iOS app 等
    🔗 https://x.com/bcherny/status/2063792263067754658
