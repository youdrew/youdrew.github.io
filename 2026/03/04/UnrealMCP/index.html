UnrealMCP - AI控制Unreal Engine的MCP项目汇总

更新时间：2026-03-04

MCP (Model Context Protocol) 是让 AI 助手直接控制 Unreal Engine 的桥梁。以下是目前活跃维护的优秀 UnrealMCP 项目。

BTW：

  • Cluade CLI配置文件路径： C:\Users\Eugene\.claude.json
  • Rider的Mcp配置文件的位置：C:\Users\Eugene\AppData\Local\github-copilot\intellij\mcp.json

🔥 推荐项目

1. GenOrca/unreal-mcp ⭐ 32 stars

  • 项目地址: https://github.com/GenOrca/unreal-mcp
  • 语言: Python
  • UE版本: 5.6+
  • 维护状态: ✅ 活跃维护（最后更新：2026-02-28）
  • 特点:
    • 完整的 Actor 操作（生成、复制、变换、删除）
    • 资产管理（搜索、过滤、静态网格详情）
    • 材质系统（参数、表达式、重编译）
    • Blueprint 节点检查
    • 行为树和黑板管理
    • 包含演示视频和截图
  • 安装方式: Git clone 或 Fab 商店
  • 适合: 需要完整 Python 工具链的团队

2. ChiR24/Unreal_mcp ⭐ 307 stars

  • 项目地址: https://github.com/ChiR24/Unreal_mcp
  • 语言: TypeScript + C++
  • UE版本: 5.0-5.7
  • 维护状态: ✅ 活跃维护（最后更新：2026-02-27）
  • 特点:
    • 原生 C++ Automation Bridge 插件
    • 支持 NPM 包安装 (`npx unreal-engine-mcp-server`)
    • 动态类型发现
    • GraphQL API 支持
    • Docker 支持
    • 44+ 工具（资产、Actor、编辑器、关卡、动画、物理、视觉特效、Sequencer、音频等）
    • 命令安全验证
    • 详细的文档和社区讨论
  • 安装方式: NPM 或 Git clone
  • 适合: 需要企业级功能和 TypeScript 生态的团队

3. etelyatn/UnrealCortex ⭐ 3 stars

  • 项目地址: https://github.com/etelyatn/UnrealCortex
  • 语言: C++ + Python
  • UE版本: 5.6+
  • 维护状态: ✅ 活跃维护（最后更新：2026-02-27）
  • 特点:
    • 模块化架构（10 个领域模块）
    • CortexData: DataTables、GameplayTags、DataAssets、CurveTables
    • CortexBlueprint + CortexGraph: Blueprint 创建、修改、编译、迁移到 C++
    • CortexUMG: UMG 控件层级构建
    • CortexMaterial: 材质、实例、参数集合
    • CortexLevel: Actor 放置、变换、组件、流式加载
    • 完整的撤销支持
    • 自动响应截断
  • 状态: v0.1.0 Beta
  • 适合: 需要深度 Blueprint 和 C++ 迁移能力的团队

📦 其他值得关注的项目

  • 项目地址: https://github.com/naming-sense/ns-unreal-mcp
  • 语言: C++ + Python
  • 维护状态: ✅ 活跃维护（最后更新：2026-02-27）

    • 支持 Codex CLI、Claude Code、Cursor、GitHub Copilot
    • 多 UE 实例选择器
    • WSL/Windows 端点回退
    • 事件流、指标、日志
  • 适合: 需要跨平台支持和多客户端集成的团队

5. mcp-tool-shop-org/game-dev-mcp

  • 项目地址: https://github.com/mcp-tool-shop-org/game-dev-mcp
  • 语言: TypeScript
  • UE版本: 5.4+
  • 维护状态: ✅ 活跃维护（最后更新：2026-02-27）
  • 特点:
    • 使用 UE5 内置的 Remote Control API（无需第三方插件）
    • 无需 C++ 编译

    • 多语言文档（中文、日文、西班牙语等）
    • NPM 包安装
  • 适合: 快速上手，不想编译 C++ 插件的团队

6. Codeturion/unreal-api-mcp

  • 项目地址: https://github.com/Codeturion/unreal-api-mcp
  • 语言: Python
  • 维护状态: ✅ 活跃维护（最后更新：2026-02-27）
  • 特点:
    • 即时、准确的 UE API 查询
    • 节省 token 和上下文
    • 减少 AI 幻觉
  • 适合: 需要频繁查询 UE API 文档的开发者

7. Flux-Point-Studios/unreal-mcp

  • 项目地址: https://github.com/Flux-Point-Studios/unreal-mcp
  • 语言: C++
  • 维护状态: ✅ 活跃维护（最后更新：2026-02-27）
  • 特点: AI 驱动的 UE 自动化

8. TrishynVolodymyr/unreal-mcp（已测试🆗）

  • 项目地址: https://github.com/TrishynVolodymyr/unreal-mcp
  • 说明: 目前使用的是TrishynVolodymyr / unreal-mcp版本的MCP。chongdashu/unreal-mcp版本的这个作者长期没有更新了。这个版本的MCP是Unreal5.7的，跑通了测试了一下，能实现很多基础功能，功能很全，基于python脚本来实现很多功能。如果是5.7的项目，再优化一下配置流程就基本上是可用级别的了。但是我尝试将这个项目改成5.6的项目失败了，有大量的接口工作需要去修改。

🎯 选择建议

需求 推荐项目
最简单上手 mcp-tool-shop-org/game-dev-mcp（无需插件，启用 Remote Control API 即可）
企业级功能 ChiR24/Unreal_mcp（NPM 包、Docker、GraphQL）
UE 5.6+ 新特性 GenOrca/unreal-mcp 或 etelyatn/UnrealCortex
Blueprint → C++ 迁移 etelyatn/UnrealCortex
多客户端集成 naming-sense/ns-unreal-mcp
API 文档查询 Codeturion/unreal-api-mcp
