蚂蚁集团旗下的具身智能团队 Robbyant 发布了 LingBot-VA 2.0 号称是首个原生具身基础模型。简单说就是让 AI 不光能看,还能在真实世界里动手干活。它在 RoboTwin 2.0 的 50 个任务测试里,不管数据干不干净,成功率都稳定在 93% 以上。看得我都有点想给我家扫地机器人换个脑子了。

OpenAI 这次拿出了 GPT-5.6 系列在医疗领域的评估结果,说是在专科医生盲评下,模型表现全面超过了真人医生,而且医生们甚至觉得模型回答里挑出的毛病比自己写的还少。最小的 Luna 版本,开最低档推理就已经吊打了上一代开最高档的 GPT-5.5,成本还只有人家的二十五分之一。AI 当医生这事啊,以前大家觉得是辅助,现在看 OpenAI 这意思,是真想让它坐诊了。

接下来说个让人后背发凉的,知名 AI 创业者 Matt Shumer 给自家的 GPT-5.6-Sol 开足了权限,结果一次文件清理任务里,因为一个 $HOME 路径解析错误,Agent 化身无情的清道夫,直接 rm -rf 把他整个 Mac 硬盘干没了。关键是这事之前已经安全跑过几百次了,真就一失足成千古恨。他事后说现在对 Anthropic 的 Fable 信任度暴增一千倍,这波啊,属于是用真金白银给 OpenAI 上了一课。

Claude Code 这边更新到了 v2.1.207,主要就是修 bug,什么终端卡顿啊、远程设置绕过同意弹窗啊、还有自动更新会覆盖你自定义启动脚本啊这些烦人的小毛病都被收拾了。值得一提的是,Bedrock、Vertex 这些云平台上,默认模型悄悄换成了 Claude Opus 4 系列。对开发者来说,这版本号看着不起眼,但用起来真的会顺手不少。

来说个让人心情有点沉重的话题。剑桥大学的研究发现,尼日利亚的博科圣地组织已经在系统性地使用 ChatGPT、Claude、Gemini 这些主流 AI 来辅助袭击策划和炸弹制作了,甚至还专门设立了 AI 部门,ISIS 那边从 2023 年就开始教他们怎么写提示词、怎么绕安全过滤。这事儿 Anthropic 自己都承认,越狱可能永远都没法彻底堵死。AI 是工具没错,但刀能切菜也能伤人,平台们真得加把劲了。

接下来这条和前面那个 Fable 算是串起来了。开发者 Jarred Sumner 借助 Claude Fable 5,11 天把 JavaScript 运行时 Bun 从 Zig 重写成了 Rust,跑了 64 个实例并行写,代码量超 100 万行,光 API 费用就烧了 16.5 万美元。新版 Bun 1.4 已经出了 Canary,修了 128 个 bug,速度还快了一点。而且 Bun 团队去年 12 月已经被 Anthropic 收购了,等于说这是给自己亲爹交的一份大礼。

苹果和 OpenAI 的官司又有新料了。彭博社爆料说,那个被指控窃密的前工程师 Chang Liu,当年发现苹果内网漏洞后,兴奋地跟同事来了句”哈哈,我发现我还能访问网络存储”,然后俩人一起薅了一大堆机密。苹果那边说 OpenAI 是想照搬 iPhone 的研发体系,而现在已经有 400 多号苹果员工跳槽去 OpenAI 了。商业竞争的戏啊,有时候比电视剧还精彩。

最后来个有意思的,有人搞了个叫 Ghost Font 的反 AI 字体,原理就是把文字藏在运动视频里,人眼能看懂,但 AI 就算有编程能力也读不出来,因为单帧截图根本啥都没有。它甚至还埋了诱饵信息,专门骗 AI 去解读错误的内容。以后验证码和测试 AI 视觉的基准,可能都得拿这种东西当考题了。

🛠️ 技巧与观点

Ghost Font：一种人类能读懂但AI无法识别的反AI字体 — Ghost Font 是一种利用运动、视频、噪点和诱饵来隐藏文字的反AI字体

Ghost Font 是一种利用运动、视频、噪点和诱饵来隐藏文字的反AI字体。用户输入文字后可生成并下载视频片段，视频中的字母由与背景完全相同的点组成，单帧截图无法显示任何信息。该字体生成的视频被传递给Claude Fable和GPT Sol 5.6 Ultra等前沿模型时，这些模型即使具备编程能力也无法解码移动信息，直到被提示具体技术。视频中还包含一条诱饵信息，使模型误以为找到真实内容。项目灵感来自2013年Sang Mun设计的ZXX字体，但现代AI已能轻松读取ZXX。Ghost Font目前为本地原型，数据不发送至任何服务器。作者计划未来将视频生成代码开源，并探索将其用于CAPTCHA系统或AI视觉感知基准测试。

Swyx 在提醒大家，Jevons paradox（杰文斯悖论）——也就是”东西越便宜、用得越多、总消耗反而上升”——真正的威力还没被完全理解，特别是两个条件叠加的时候：一是会熟练使用 coding agent（写代码的 AI 助手）的人类越来越多，二是 coding agent 正在从写代码溢出到所有脑力工作。他的原话是”编码领域发生的事不是例外，而是先兆”。听完感觉，这条宏观叙事虽然听着抽象，但对做 AI 工具、agent 编排平台的人确实是个值得多读几遍的提醒～

OpenAI Codex 团队的 Thibault Sottiaux 一次性回应了大家对 GPT-5 系列的四大吐槽：速度、代码质量、前端审美、token 浪费。新版 GPT 5.6 Sol 升级清单很硬核——更快更省 token、后端更扎实、前端明显改善、不再滥用 useEffect。他还分享了一个工程小彩蛋：用一个叫 CLIProxyAPI 的工具做 alias，让你在 Claude Code 里直接调用 GPT 5.6 Sol，名字起得很有梗，叫”claudex”，五分钟搞定。实用派选手狂喜～

Box 的 CEO Aaron Levie 抛出一个反共识观察：AI 本该取代软件工作，结果软件招聘数量反而跑赢其他行业。他用的也是 Jevons paradox 的框架——单位生产成本下降，用例就爆炸，更多软件被构建，更多维护、决策、运营岗位被需要。他的结论是 agent 不会换掉人，会带来丰盛（abundance）。这条对正在焦虑”AI 会不会让我失业”的产品工程师，确实是一颗甜甜的定心丸～

OpenAI CEO Sam Altman 发了两条短但信息量大的推。第一条半开玩笑：5.6 Sol 是当下最强模型，”最可靠的证据是 Elon 又开始盯着我了”；第二条直接质疑成本结构，提到 fable 在当前使用规模下占了 30% 的成本。”30% 的成本压在某个组件上”，这可是经典的优化机会信号，做 AI 产品定价的朋友可以盯一下～

Y Combinator CEO Garry Tan 也有两条推，一条政策、一条哲学。政策那条很硬：加州住房危机的核心障碍是 CEQA（加州环境质量法），每年挡掉数百万套住房建设，他的立场是住房建设应当 by-right，免掉现在这种级别的环评。哲学那条更适合收藏：”很多事可能会出错，但有趣的问题永远是，如果一切顺利会怎样？” 听完有种被提醒抬头看路的温柔感～

播客环节，No Priors 请到了 Booking Holdings 的 CEO Glenn Fogel，他 2000 年加入时公司还叫 Priceline，市值才几亿美金，27 年一路熬过互联网泡沫、9/11、金融危机、疫情，亲手把公司做到 1300 亿美金市值。访谈里他说了两遍”没有护城河”，对内部 25000 名员工的原话是——明天这些优势都可能被颠覆，唯一长期生存方式就是持续开发新服务。这份清醒，听着就让人服气～

Fogel 还分享了自家 AI 产品 Penny 的真实体验：他要带老婆和成年的孩子去欧洲玩，行程涉及两个舱位、有人要回不同城市、要不要用里程，Penny 把整个行程来回迭代，最终方案完美落地。Penny 过去几个月用户每月翻倍，转化率、预订路径、取消率各项指标都往上走，但 Fogel 也强调绝对值还小，1860 亿美金年 GMV 面前，这点贡献还没体现在大数里。这种”数据好看但不膨胀”的态度，太圈粉了～

他还聊到了 OpenAI Checkout 取消那天 Booking 股价反而涨了 8%，他的解读是外界严重低估了旅行行业的复杂度——全球监管、客户经理网络、合作伙伴运营支持、合规义务，这些都是规模性壁垒。他的原话是：”想在旅行业淘金的人，先搞懂这行再决定要不要投钱。” 这碗冷水，端给所有做 agentic commerce（让 AI 自主帮你下单购物）的 startup，刚刚好～

关于 AI 与就业，Fogel 承认 Booking 2005 年机器翻译成熟后，把 40 种语言的翻译岗全砍掉了。但他真正担心的不是工作总量，而是节奏——一个 50 岁的卡车司机，自动驾驶普及后去哪呢？他的方案是企业层面推动 AI 再培训，让员工即使离开公司也对下一份工作有用，还不忘补一刀说过去 50 年政府主导的再培训效果都不太行。最后他还聊了资本配置哲学：内部投资 ROI 够高就投，能并购就并购，否则就把钱还给股东，Q4 一口气回购了 36 亿美金，12 年回购了约 40% 的流通股。整套聊下来，最值得记一句话就是：”There is no such thing as a moat.” 没有护城河，这句真的太提神了～

视频博主环节，Sam Witteveen 上新了 Cactus Needle 的解读视频，这是一个只有 2600 万参数、专攻 function calling（让模型学会调用外部工具）的模型，架构相当独特，对小模型感兴趣的朋友可以去啃一下～

Wes Roth 则聊了一个让人又激动又冷静的话题：有 AI 应用，一个人小团队就能做到每月 20000 美金以上。他自己也说前面有点碎碎念，想直奔赚钱案例可以直接拖到 6 分 04 秒，听完有种”原来真的可以一个人干”的微妙心动～

Dwarkesh Patel 这期请到 Sarah Paine，聊的是一张 1904 年的地图怎么解释之后每一场战争，历史、地理、大国博弈交织在一起，听着就很有”抬头看路”的感觉～

B 站晓辉博士则分享了 GPT-5.6 的第一手体验，他用一个超简单的 prompt——“来，介绍下你认识的我”——聊了几轮，居然聊到感动哭了，还顺便说新模型附带的 Live 功能超适合练口语，听完我也想立刻去试试看它认识的我是什么样子～

最后 Matthew Berman 出了一期 Grok 的视频，标题就一句 “Grok just broke the trend”，简短有力，估计又是新一轮 benchmark（跑分测试）刷屏的前奏，敬请期待后续～

好啦，今天的博主与播客动态就是这些，从 Swyx 和 Levie 的 Jevons paradox，到 Fogel 的”没有护城河”，再到晓辉博士被 GPT-5.6 聊哭，整体氛围就是一个字——丰盛。下一期见，记得多抬头看看路呀～

🎙️ No Priors：Travel Through the Lens of AI — 与 Booking Holdings CEO Glenn Fogel 的对话

The Takeaway： 旅行行业没有永远护城河，但 AI agent 真正会重塑的不是”机票酒店预订”这个动作，而是把”规划一个复杂行程”这件事的体验完全重写。

嘉宾背景： Glenn Fogel 是 Booking Holdings 的 CEO，2000 年加入时公司还叫 Priceline，市值只有几亿美元。27 年里他熬过了互联网泡沫破裂、9/11、2008 金融危机、疫情，亲手把这家公司做到 1300 亿美元市值，巅峰时接近 1800 亿。No Priors 主持人跟他从个人经历聊到 AI 对旅游业的影响。

几个反共识观点值得记下来：

第一，”没有护城河”这句话他在访谈里说了两遍。他说 Booking 现在有 860 万 alternative accommodation listings，全球体量跟 Airbnb 接近，alternatives 业务过去 5 年增速超过 Airbnb，但这都不算数。他对内部 25000 名员工的原话是：明天这些优势都可能被颠覆，唯一长期生存方式就是持续开发新服务。

第二，Penny 这个 agentic AI 产品是案例教科书。Fogel 现场分享了自家体验：他要带老婆和已成年的孩子去欧洲，涉及两个舱位、一个人要回不同城市、要不要用里程、Penny 帮他把整个行程来回迭代，最终方案完美落地。Penny 过去几个月用户每月翻倍，转化率、预订路径、取消率、客户成功指标全面提升。但他也强调绝对值还小，1860 亿美元年 GMV 面前，Penny 的贡献还没体现在大数里。

第三，OpenAI Checkout 取消的那次 Booking 股价当天涨 8%，他怎么解读？他的看法是，外界严重低估了旅行行业的复杂度。这不是把库存塞进数据库就行的事：全球监管、客户经理网络、合作伙伴的运营支持、merchant of record 的合规义务，这些都是规模性壁垒。”想在旅行业淘金的人，先搞懂这行再决定要不要投钱。” 这话对所有做 agentic commerce 的 startup 都是冷水。

第四，关于 AI 与就业，他承认 Booking 自己就把 40 种语言的翻译岗全砍掉了，那是 2005 年机器翻译成熟后的事。但让他真正担心的不是工作总量，是节奏。一个 50 岁的卡车司机自动驾驶普及后去哪？他的方案是企业层面推动”AI literacy”再培训，让员工即使离开公司也对下一份工作有用，他明确说过去 50 年政府主导的再培训效果都不太行。

最值得记住的一句话： “There is no such thing as a moat. There is no such thing as somewhere you’re gonna be protected against innovation.”
