PagedWeight: Efficient MoE LLM Serving with Dynamic Quality-Aware Weight Quantization

PagedWeight 针对的是当前大模型部署中一类棘手矛盾：MoE（混合专家）架构虽因按需激活专家而具备较高计算效率，但在长上下文推理场景下，KV 缓存会持续膨胀，与庞大的专家权重争夺有限的 GPU 显存，传统静态分配方案难以兼顾两侧需求。为此，作者提出一种面向 MoE 推理服务的权重管理方法，其核心思路是在运行时对专家权重进行动态量化，并以"质量感知"的方式在不同请求之间灵活调配精度。具体而言，PagedWeight 会暴露并主动权衡任务精度、显存占用与吞吐量三者之间的关系：当 KV 缓存压力上升时，系统可降低部分非关键专家的权重量化比特以释放显存；当压力缓解时，再相应恢复精度，从而在保证任务表现的前提下提升整体吞吐。该方案的意义在于突破了 MoE 模型权重与 KV 缓存之间此消彼长的二元取舍，为长上下文、高并发的 LLM 在线服务提供了一种更细粒度的资源调度思路，对降低显存门槛、提升单位硬件下的服务规模具有实际参考价值。
