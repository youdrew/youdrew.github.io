美团把自家的大模型 LongCat-2.0 全部开源了,而且是 MIT 许可,意思就是随便用、随便改、随便商用,门槛几乎为零。这个模型有 1.6 万亿参数,每次只激活大约 480 亿,主打一个长文本——能一口气吃下 100 万 token 的上下文。它在几个硬核代码测试里直接超过了 GPT-5.5,听着是不是有点上头?国内团队做出这种水平的开源模型,真的可以骄傲一下。

接下来是 Meta 的大动作。扎克伯格亲口说,他们在砸数千亿美元建一个叫 Prometheus 的 AI 集群,单个集群规模超过一个吉瓦,这是什么概念?基本就是 AI 时代的”超级发电厂”,专门给大模型供电。听他这意思,就是把人才、资金、电力全押上,赌的就是 AGI 这一票。

然后是个不太好的消息。NVIDIA 那个超酷的下一代服务器架构 Kyber NVL144,本来预计很快面世,结果直接跳票到 2028 年,延期超过一年。连带着另一个 NVL72x2 的机架设计也被砍了。受影响最大的就是 Rubin Ultra 这代芯片,扩展能力会被卡住。看来 AI 硬件这块,谁也不敢保证按时交货。

说个让人有点担心的事。欧盟理事会用快速通道悄悄通过了一项叫”Chat Control 2.0”的法规,要求科技公司对所有人的加密聊天内容进行扫描。批评者说这是绕过了正常的民主监督,趁着夏天议会休会想硬塞过去。表面说只扫必要内容、扫完一年就删,但加密聊天的意义不就是为了不被扫吗?这事后续值得盯着。

开源圈也有新动作。Anthropic 旗下的 Claude Design,有人把它背后的系统提示词反编译出来,整理成 20 章提示词加 14 项技能,放到 GitHub 上 MIT 协议开源了,还顺带更新支持了最新的模型。里面从美学、无障碍到交互状态都讲得很细,前端设计师和 AI 玩家可以拿去当参考,相当于把官方”魔法咒语”扒给你看。

做法律 AI 的朋友可以关注一下 LlamaIndex 新出的 legal-kb,这是一个法律文档知识库的参考应用。它给 AI Agent 配了四个像”找文件””读文件””搜内容”这种操作系统风格的工具,让 Agent 能像人查资料一样去翻法律文档,而不是一股脑全塞进上下文。对做企业知识库或者 RAG(也就是让 AI 准确检索资料的技术)的人来说,是个不错的参考样板。

最后讲个有点暖的故事。一位爸爸为了不会说话的自闭症儿子,做了一款沟通 App,结果在治疗室被其他家长看到,妈妈们当场哭了,治疗师也哭了五分钟。三周前他还是个普通上班族,现在意外发现真的有人需要这个产品,决定咬牙全职做下去。这个 App 跟传统的辅助沟通设备不一样,专门为”听得懂但说不出来”的孩子设计——这个细分人群其实长期被忽略,有人愿意为他们花时间,本身就挺动人的。

Nan Yu（Linear 产品负责人）这批在反思 coding agent 时代的工程判断。两句金句必须记一下：一句是”你把生产环境表 drop 干净，到底是模型被开除还是你被开除”——精准戳中 agent 出错后的责任归属灰色地带；另一句更搞笑：”以前手动写代码进入 flow 状态我就会持续爆粗口，所以 coding agent 基本就是 AGI”（自嘲版）。但最有长期价值的是他调试 bug 的方法论：别死磕代码找 bug，要真的去用产品、把它用到挂；code review 的精力应该放在架构和 API 设计上，因为后者才是技术债的真正源头。

Cat Wu（AnthropicAI 的 Claude Code + Cowork 负责人，前 Dagster / Scale AI）这批给 Fable 5 交了份很实在的测评。她做了个真实场景测试：让 Fable 5 跑留存分析，结果它自己主动选了 propensity score matching（也就是倾向性得分匹配——按用户行为维度先把相似用户配对、再做对照分析，这样比较有说服力）——她根本没提示。Fable 5 这种”挑对方法”的本能明显变强，从 Cowork 里写邮件写文档，到 Claude Code 里 debug 复杂 bug，judgment 是真的全面提升了。模型有了”品味”，大家干活都松口气。

Guillermo Rauch（Vercel CEO）这批放了个真值得看的数据观察：他把 Vercel AI Gateway（Vercel 的统一 LLM API 代理网关，帮你路由请求、聚合多模型用量）的全生命周期 token 支出做成了一段动画，数据来自每月几百万开发者、合计好几万亿 token。两个结论：Anthropic 在路由份额上依然遥遥领先，开源权重模型的占比明显抬头。要是你正在琢磨”应用层底下到底是哪几家粘性强”，这条动画必须盯一会儿。

Peter Steinberger（自称 “Polyagentmorous ClawFather”，同时在 OpenClaw 和 OpenAI 两边跑）这批又在交付东西了。值得关注的一条：他预告了自己那个 Claude Code 风格工具的下一版本，能直接看到 session 的 reset 时间，方便你玩 token-maxxing——也就是把单次 session 的 token 配额刷满，是大配额、长上下文场景下的极致压榨玩法。另一条是 “what a ride” 配上转推庆祝，氛围向。核心钩子就是新功能上线，记得去推里翻链接。

Dan Shipper（Every CEO）这批回到”我在干活”的轻节奏。两条都挺有意思：一条是吐槽 ChatGPT 上线 Codex-in-ChatGPT（”Yo dawg，听说你喜欢 Codex，我们在 ChatGPT 里又套了一层 Codex”）——角度刁钻，因为 Every 自己也做 Codex 相邻产品，而 OpenAI 正在自己平台里加这个方向的 wrapper；另一条是轻安利 3blue1brown 上 Dwarkesh 那一期访谈，顺便自嘲”原来我著名数学家名字的发音一直都错”。整体是淡的本，适合下午茶配着听。

Anthropic 这批在官方博客里发了个对 Apple 开发者很友好的更新：出了一个 Swift package，让你能从 Apple 的 Foundation Models 框架里直接调 Claude。核心玩法很优雅——比如一个 journal app，先用 Apple 的端侧模型生成每日的反思 prompt，再把”整月笔记找主线”这一棒交给 Claude；学习 app 也可以先在端侧把术语定义出来，学生追问”那这和前面讲的所有东西有什么关系”时再甩给 Claude。对开发者最实在的好处是：Foundation Models 框架那三行 API 拿到的 typed Swift values（带类型的 Swift 结构体返回值），Claude 接力做完多步推理、写代码、搜索或执行代码，再把结构化结果流回同一个 SwiftUI 视图。iOS 27、iPadOS 27、macOS 27、visionOS 27、watchOS 27 “明天”上线，装好 package、配一把 Anthropic API key 就能双向路由。

Cloudflare CEO Matthew Prince 上 The MAD Podcast 抛出个相当硬的核心判断：Cloudflare Radar 数据显示，2026 年上半年 bot 流量正式超过人类流量——比他们自己 2025 年的预测整整提前了五年。按这个速度，未来 5 年内互联网流量会增长 1000 倍，更多服务器、GPU、内存全在伺候 agent。Prince 直接说”由于 bot 不点广告，过去 28 年那条广告模式已经死了”，得有人发明下一个我们还说不出来的东西。Cloudflare 自己产品线就是这条主线最具体的注脚：Workers（基于 isolate 的 serverless 运行时——serverless 就是开发者不管服务器、只写函数逻辑、平台按需起算力；isolate 是 V8 引擎里的轻量隔离沙箱，比传统 VM/Container 启动快、开销小，专门为短时高频 agent 设计）天然契合”agent 一直在写代码”的场景；AI Gateway 做审计、prompt injection 防御、按成本路由 OpenAI / Anthropic 等多模型，OpenAI 移动 app 和 Anthropic 的 cloud.ai 都跑在他们上面；新一套装了 Dart/Rust/GPU 的边缘基础设施，专门为”每个知识工作者哪怕只配一个 agent、CPU 需求暴增 40 倍”这种冲击做准备。最让人心头一紧的是安全那条线：Prince 预计未来两年每周都会冒出一个 Log4j 级漏洞，但接着会急转弯——用 Cloudflare 十年事故数据训练、和团队偏见”不相关”的 AI code review agent 会让软件质量大幅跃升，他内部事故 noise 曲线已经掉下了一个断崖。Cloudflare 的发家史本身也是堂”靠副产品定价”的经典课：他们当年的”土耳其 escort 网站群”和”Hacker Kids”那批用户（”世界上的黑客小子一夜之间全注册了”），在 Cloudflare 的 ML 系统里被打上”TE”标签——结果这套攻击模式背后的同一个人，从当年的伊朗学生一路打到阿塞拜疆 Eurovision、再打到美国几家大银行，整条攻击者职业曲线能被 Cloudflare 看清，就是因为它愿意从”互联网这些怪角落”里学习。听完这期，广告业务的同学可能需要更新一下自己的五年规划了。
