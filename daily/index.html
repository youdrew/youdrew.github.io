Cat Wu，Anthropic 的 Claude Code 和 Cowork 负责人，前 Dagster 和 Scale AI 员工，发了一条很实在的 Fable 5 测评。她做了个真实场景测试：让 Fable 5 跑留存分析，它自己主动选了 propensity score matching，也就是”倾向性得分匹配”——按用户行为维度先把相似用户配对再对照分析，这样比较有说服力，她根本没提示。她把这件事解读为 Fable 5 的 judgment 全面提升：从 Cowork 里写邮件写文档，到 Claude Code 里 debug 复杂错误，模型”挑对方法”的本能都明显变强了。

Anthropic 官方博客：出了一个 Swift package，让 Apple 开发者能从 Apple 的 Foundation Models 框架里直接调 Claude。核心玩法很优雅：一个 journal app 可以先在端侧用 Apple 模型生成每日反思 prompt，再把整月笔记做”主题串联”这一步甩给 Claude；一个学习 app 可以先在端侧把术语定义出来，学生追问”那这和前面讲的所有东西有什么关系？”时再交给 Claude。Foundation Models 框架本身能用三行代码拿到带类型的 Swift 结构体返回值，Claude 这边接力过去、做完多步推理、写代码、搜索、执行代码，再把结构化结果流回同一个 SwiftUI 视图。iOS 27、iPadOS 27、macOS 27、visionOS 27、watchOS 27”明天”上线——把 package 装好、配一把 Anthropic API key，就能双向路由了。

“加了 Claude 之后这些模式都被延伸：比如 journal app 可以端侧生成每日反思 prompt，再让 Claude 跨月份笔记里找主线。”

来源：https://claude.com/blog/claude-for-foundation-models

播客方面，Cloudflare 的 Matthew Prince 在 MAD Podcast 上扔出了一个里程碑级数据：Cloudflare Radar 显示 2026 年上半年 bot 流量正式超过人类流量，比他们自己 2025 年预测的”2027 年底”提前了五年。他的核心判断很硬：按这个速度未来 5 年内流量会增长 1000 倍——更多服务器、GPU、内存，全部为了伺候 agent。而 bot 不点广告，所以过去 28 年靠广告撑起来的互联网商业模式已经”死了”，得有人发明下一个我们还说不出来的东西，连”品牌”在 agent 世界里的含义都要重写。

Cloudflare 自己的产品线就是这条主线最具体的注脚：Workers 是基于 isolate 的 serverless 运行时，”serverless”就是开发者不用管服务器、只写函数和逻辑，平台按需起算力实例；”isolate”是 V8 引擎里的轻量隔离沙箱，比传统 VM/Container 启动快、开销小，专门为短时、高频 agent 工作负载设计，特别契合”agent 一直在写代码”的场景。AI Gateway 做审计、prompt injection 防护、按成本智能路由 OpenAI 和 Anthropic 等模型，OpenAI 的移动 app 完全跑在 Cloudflare 上，Anthropic 的 cloud.ai 也叠在它上面。再加上一套装了 Dart、Rust、GPU 的新边缘基础设施，专门为”每个知识工作者哪怕只配一个 agent、CPU 需求就要暴增 40 倍”这种冲击做准备。

最让人心头一紧的是安全这条线。Prince 预计未来两年每周都会冒出一个 Log4j 级漏洞，因为 AI 模型找漏洞的能力已经”疯癫”；但接着会急转弯——“和团队偏见不相关”的 AI code review agent，用 Cloudflare 十年事故数据训练，会让软件质量大幅跃升。他内部已经看到这事发生：AI code review 上线那一刻，他们的事故 noise 曲线直接掉下了一个断崖。

Cloudflare 的发家史本身也是一堂经典课：第一批”等价付费”用户是一堆土耳其 escort 网站和 Hacker Kids，套攻击模式在 ML 系统里被打上”TE”也就是 Turkish Escort 标签。后来故事更诡异——同一个人，最初是伊朗的一个学生，后来又去打了阿塞拜疆巴库办的 Eurovision，又去打了美国几家大银行。Cloudflare 之所以能看到这条完整的攻击者职业曲线，就是因为它愿意从”互联网这些怪角落”里学。

“2026 年上半年 bot 流量正式超过人类流量……5 年内今天的互联网流量可能会变成 1000 倍。”
“过去 28 年互联网的基本商业模式基本上没变，就是广告。问题是 bot 不点广告。”
