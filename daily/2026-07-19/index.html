09Claude Code

Claude Code v2.1.215 更新

Claude Code 推出 v2.1.215 版本更新，主要调整涉及两项开发辅助技能的行为触发机制。具体而言，在该版本之前，/verify 和 /code-review 两个技能可由 Claude 在合适场景下自主决定是否调用；在 v2.1.215 中，这一自动触发行为被取消，两个技能改为仅在用户主动输入指令时才会执行——用户需显式输入 /verify 触发校验流程，或输入 /code-review 触发代码审查流程。此次改动将技能调用权完全交还给用户，避免了模型在不需要时自动介入代码校验或审查而造成额外耗时与打扰。从影响上看，这是一次偏向精细化交互控制的体验调整，对追求自动化建议的开发者意味着需要更主动地发出指令，但同时提升了操作的可预期性，让用户能根据实际需要选择何时引入额外的验证与审查步骤；该版本未涉及功能新增、性能优化或其他重大能力变更，属于一次轻量级的行为策略迭代。
